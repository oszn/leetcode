[50. Pow(x, n)](https://leetcode.cn/problems/powx-n/)

快速幂求解。

```c++
    double myPow(double x, int n) {
        double ans=1;
        if(x==1){
            return x;
        }
        
        if(n==0){
            return 1;
        }
        if(n<0){
            x=1/x;
        }
        while(n!=0){
            if(n%2){
                ans*=x;
            }
            x=x*x;
            n=n/2;
        }
        return ans;
    }
```
