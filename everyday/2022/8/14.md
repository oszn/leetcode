[1422. 分割字符串的最大得分](https://leetcode.cn/problems/maximum-score-after-splitting-a-string/)

记住右边1的数目，然后从0开始，非常easy。

```c++
  int maxScore(string s) {
        int cout_one=0;
        int n=s.size();
        for(int i=0;i<n;i++){
            if(s[i]=='1'){
                cout_one++;
            }
        }
        int max_step=0;
        int scores=cout_one;
        for(int i=0;i<n-1;i++){
            if(s[i]=='0'){
                scores++;
            }else{
                scores--;
            }
            max_step=max(scores,max_step);
        }
        return max_step;
    }
```
